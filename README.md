# 🌤️ Weather App

A simple and cross-platform weather application built with **Flutter**. This project enables users to view **real-time weather information** for any city and runs seamlessly on **Android, iOS, Web, Windows, macOS, and Linux**.

## 🚀 Features

- 🔄 Real-time weather data for user-specified locations  
- 📱 Multi-platform support: mobile, web, and desktop  
- 🧼 Clean and intuitive UI with modern Flutter widgets  
- ☁️ Weather info including temperature, humidity, and wind forecasts  
- 🌐 Integrates with [OpenWeatherMap](https://openweathermap.org/)
